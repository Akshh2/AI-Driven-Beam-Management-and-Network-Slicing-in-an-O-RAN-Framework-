# AI-Driven-Beam-Management-and-Network-Slicing-in-an-O-RAN-Framework-
An AI-driven O-RAN smart city simulation in MATLAB, integrating Q-learning for dynamic PRB allocation and DQN-based beamforming. Modeled eMBB, URLLC, and mMTC slices with realistic channel conditions, achieving higher throughput, lower latency, improved SINR, and better fairness than baseline.
